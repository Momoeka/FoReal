Foreal Project Overview

Welcome to Foreal! This application is designed to demonstrate the integration of various modern technologies and practices to create a robust, secure, and scalable web application.

Features

  • CRUD Operations
  • Image Upload and Delete
  • Spring Security
  • Role-Based Access Control
  • Payment Integration using RazorPay
  • Session Management with Thymeleaf
  • Google Auth Login
  • Image Storage on AWS S3
  • Database Deployment on AWS RDS
  • Hosting on AWS EC2
  • Relational Database Design
  • Professional UI with Navbar, CSS, and JS
